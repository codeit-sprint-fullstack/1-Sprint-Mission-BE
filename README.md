### ✅ 환경 변수 설정

- `DATABASE_URL`: PostgreSQL 데이터베이스 연결 URL

- .env 파일에 추가(예시) : DATABASE_URL="postgresql://postgres:비밀번호@localhost:5432/데이터베이스명?schema=public"
